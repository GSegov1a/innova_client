Innova Client Demo para Windows
===============================

Contenido
---------
- instalar.bat: ejecutalo solo una vez.
- configurar_audio.bat: usalo si no se escucha, no escucha, o cambiaste microfono/parlante.
- iniciar.bat: uso normal del cliente.

Primer uso
----------
1. Descomprime el ZIP.
2. Ejecuta instalar.bat con doble click.
3. El instalador abrira la configuracion de audio.
4. Elige el numero del microfono.
5. Elige el numero del parlante o audifono.
6. Cuando termine, ejecuta iniciar.bat.

Uso normal
----------
Ejecuta iniciar.bat.

Dentro de la consola:
- Presiona c para conectar o desconectar.
- Presiona q para salir.

Si Python no esta instalado
---------------------------
Instala Python desde:
https://www.python.org/downloads/

Durante la instalacion marca:
Add Python to PATH
